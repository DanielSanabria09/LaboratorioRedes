# Simulación de Red – Cisco Packet Tracer

## Descripción del Proyecto

Este proyecto consiste en el diseño e implementación de una topología de red simulada utilizando Cisco Packet Tracer.  
La red incluye un dispositivo Cloud-PT, Home Router, Switch, PC y Server, con configuración de direccionamiento IPv4, servicio DHCP y validación de conectividad.

La documentación técnica completa del laboratorio se encuentra en la sección Wiki de este repositorio.


## Objetivos Técnicos

- Diseñar una topología de red funcional.
- Configurar direccionamiento IPv4 (LAN y WAN).
- Implementar y validar el servicio DHCP.
- Analizar el comportamiento y las limitaciones del dispositivo Cloud-PT.
- Verificar conectividad mediante pruebas de ping.
- Documentar el desarrollo del laboratorio en GitHub Wiki.


## Tecnologías Utilizadas

- Cisco Packet Tracer 9.0
- Direccionamiento IPv4
- Configuración de DHCP
- GitHub (Repositorio y Wiki)
